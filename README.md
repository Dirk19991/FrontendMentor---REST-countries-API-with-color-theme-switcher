## Полное описание

Для проекта использовался REST Countries API. Style Guide был предоставлен на [странице челленджа](https://www.frontendmentor.io/challenges/rest-countries-api-with-color-theme-switcher-5cacc469fec04111f7b848ca/hub/rest-countries-api-with-color-theme-switcher-t5xZ7hD1oW).

1. На главной странице отображается полный список стран со краткой статистикой - регион, население, столица. Список можно фильтровать по региону и по поисковому запросу,  фильтры могут быть задействованы одновременно:
<img src="https://user-images.githubusercontent.com/104031523/204243578-2e22ec84-e20a-450b-9d7a-4918aaca5615.gif" alt="Alt text" title="Optional title" width=70% height=70%>

2. На карточке каждой страны отображается более подробная информация: валюта, домен, языки и страны, граничащие с данной страной. С карточки можно переходить на карточки соседних стран:
<img src="https://user-images.githubusercontent.com/104031523/204245947-c5fe872c-a262-46cc-9515-8e4ff59ac9d7.gif" alt="Alt text" title="Optional title" width=70% height=70%>

3. Реализована адаптивная верстка:
<img src="https://user-images.githubusercontent.com/104031523/204244972-0622ae28-2434-4cff-ae6a-83f14f6c26fb.gif" alt="Alt text" title="Optional title" width=70% height=70%>

4. Реализованы темная и светлая темы:
